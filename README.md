# Zambian School Management System

Premium School Management System for Zambian schools - Next.js 14, TypeScript, Tailwind CSS, PostgreSQL

## Features
- Student, Staff, Parent Management
- Academic Structure (Grades, Classes, Subjects)
- Exams & Results
- Fee Management
- Attendance Tracking
- Library, Hostel, Transport
- SMS/Email Communication

## Tech Stack
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- PostgreSQL + Prisma
- NextAuth.js v5
